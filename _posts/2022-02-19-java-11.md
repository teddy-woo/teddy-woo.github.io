# 복수행 주석 /`` ``/

한 문장 이상의 주석을 적는데 사용하는 방식입니다.

```
/* console.log('안녕'); */

/*
console.log('안녕1');
console.log('안녕2');
*/

/*
 * console.log('안녕1');
 * console.log('안녕2');
 *
 *
 */
```
